# Platform Analysis: Dzen, Telegram, Medium, WordPress

## 1. Yandex Dzen (Яндекс Дзен)

### Характеристики
- Российская платформа контент-маркетинга
- Основана на рекомендательной системе
- Рост через автоматический алгоритм, не нужны подписчики

### Аудитория
- Русскоязычная аудитория (РФ, СНГ)
- Возраст: 25-55
- Интересы: саморазвитие, развлечения, новости

### Требования
- Минимум 0.5k просмотров в месяц для монетизации
- Рекомендуемая длина статьи: 2000-3000 символов
- Есть встроенная монетизация (доли от рекламы)

### Сильные стороны
- Встроенная монетизация
- Хороший трафик благодаря рекомендациям
- Низкий порог входа

---

## 2. Telegram

### Характеристики
- Мессенджер с каналами и группами
- Прямая коммуникация с подписчиками
- Отличная для аффилиатских ссылок и CTA

### Аудитория
- Активная, целеустремлённая аудитория
- Возраст: 18-45
- Высокая вовлеченность

### Требования
- Необходимо создать канал
- Поддержка ссылок и документов
- Можно интегрировать бот для опросов

### Сильные стороны
- Высочайшая вовлеченность
- Идеально для ссылок на аффилиатские программы
- Персональное взаимодействие

---

## 3. Medium

### Характеристики
- Платформа для длинных статей
- Международная, англоязычная аудитория
- Встроенная монетизация Member Program

### Аудитория
- Технологи, предприниматели, специалисты
- Возраст: 25-50
- Англоязычная

### Требования
- Статьи 1500+ слов
- Кросспост на русскоязычные платформы для монетизации
- Нужна уникальная ценность

### Сильные стороны
- Встроенная монетизация
- Высокий SEO
- Международная аудитория

---

## 4. WordPress (Собственный блог)

### Характеристики
- Полный контроль над контентом
- Возможность подключить рекламу (AdSense, etc.)
- SEO-оптимизированный

### Аудитория
- Зависит от нашего контента
- Органический трафик через поиск

### Требования
- Хостинг + домен
- SEO-оптимизация
- Регулярные обновления

### Сильные стороны
- Полный контроль
- Собственная аудитория
- Масштабируемость
